**Atenção:**  
Este projeto foi descontinuado e está sendo arquivado. Entretanto, ele servirá como base para uma nova implementação chamada **Cerrado Cloud**, que continuará sua evolução com melhorias e novos recursos.

Todos os direitos e créditos pelo desenvolvimento original permanecem integralmente com **Maickonn Richard** e **Gabriel Stilben**, criadores do *AllaKore Remote*.

---

## Sobre o Projeto Original

O **AllaKore Remote** é um software de acesso remoto open source desenvolvido em **Delphi Seattle**, utilizando apenas componentes nativos da própria IDE.

### Observações importantes antes de abrir o projeto:

- O software depende de um **servidor central**, preferencialmente hospedado em seu país para reduzir latência.  
- Como um projeto BETA, pode conter bugs.  
- Correções e contribuições sempre foram bem-vindas no projeto original.  
- O servidor é responsável por **rotear todo o tráfego**, encaminhando pacotes imediatamente para maior desempenho.  
- No cliente, na unit `Form_Main`, existem as constantes `Host` e `Port`, que devem ser configuradas conforme seu servidor.

---

## Funcionalidades do AllaKore Remote

- ID e Senha de Conexão  
- Acesso remoto com algoritmo **RFB** (envia apenas mudanças da tela)  
- Compressão de dados via **zLib**  
- Compartilhamento de arquivos  
- Chat integrado  
- Interação com **UAC**  
- Demos funcionais
    
---

## Nota Atualizada – Transição para Cerrado Cloud

Este código será utilizado como **base para o novo projeto _Cerrado Cloud_**, que continuará e modernizará o trabalho iniciado pelo AllaKore Remote, **mantendo integralmente os créditos aos desenvolvedores originais**.
